Design Improvement Instructions

The current design uses too many different colors. I need better color consistency across the entire website. Please simplify the color system and maintain a minimal but classy design style.

Color Consistency

Use one primary color across the website.

Use one secondary color if needed.

Use one accent color for buttons and highlights.

Avoid using too many different colors in different sections.

Maintain the same color palette across all components including:

buttons

headings

icons

cards

backgrounds

The design should feel clean, modern, minimal, and consistent.

Overall Design Style

The website should follow a minimal but elegant design.

Requirements:

clean layout

enough white space

simple typography

soft shadows

rounded cards

consistent spacing

modern UI but not overly decorative

Avoid:

excessive gradients

too many colors

heavy animations

overly complex layouts
